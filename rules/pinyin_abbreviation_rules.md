# 拼音首写转换规则说明文档

## 📋 规则概述

**规则名称**：拼音首写转换规则  
**规则ID**：`pinyin_abbreviation_rules`  
**版本**：1.1  
**作者**：系统  
**功能描述**：读取 Excel 中的【字段名称】列，为每行生成对应的拼音首字母简称，并新建【拼音简称】列；英文/数字保持并转为小写，中文按汉字取拼音首字母，结果整体为全小写。同时新增【重复】列：若该行拼音简称在表中出现多于一次，则标记为「是」，否则为「否」。

## 📥 输入要求

### Excel 文件结构

- 支持单工作表或多工作表；处理时以当前选中的工作表（或第一个工作表）为数据源。
- 文件格式：`.xlsx`。

### 必需列

- **【字段名称】**：文本类型，表示需要转换的字段名（可包含中文、英文、数字、下划线等）。

### 示例数据

| 字段名称       |
|----------------|
| 工单           |
| PMC_评审日期   |
| 创建时间       |
| 项目名称       |

## 🧮 转换逻辑详解

### 基本规则

1. **结果列**：在原有数据基础上新建【拼音简称】列，与【字段名称】逐行对应；新建【重复】列，标记该行拼音简称是否在表中重复。
2. **大小写**：拼音简称**全部为小写**。
3. **分段规则**：以**下划线 `_`** 为分隔符，将字段名称拆成多段，每段分别转换后再用下划线连接。

### 每段转换规则

对「字段名称」或其中的每一段（以下统称「当前段」）：

- **仅包含英文字母、数字等非中文字符**  
  - 整段转为小写，原样保留（仅做小写化）。  
  - 例：`PMC` → `pmc`，`ABC123` → `abc123`。

- **包含中文字符**  
  - 对每个汉字取其**拼音的首字母**（小写），依次拼接。  
  - 例：`工单` → 工(gong) 单(dan) → `gd`；`评审日期` → `psrq`。

- **中英混合段**  
  - 按字符顺序：英文字母/数字 → 转小写并保留；每个汉字 → 取其拼音首字母（小写）。  
  - 例：`评PMC审` → `ppmc`（评→p，PMC→pmc，审→s，若按字序则为 p、p、m、c、s，此处示例按「英文整体小写+汉字首字母」理解为 `p`+`pmc`+`s` → `ppmc`）。  
  - 若业务要求「英文整体保留、汉字取首字母」，则按此执行；若要求「逐字符：英文保留、汉字取首字母」，在实现时可在 py 中二选一并在此说明。

### 连接与最终结果

- 多段之间用**下划线 `_`** 连接。  
- 最终【拼音简称】为**全小写**字符串。

### 转换示例

| 字段名称       | 拼音简称   | 说明 |
|----------------|------------|------|
| 工单           | gd         | 工→g，单→d |
| PMC_评审日期   | pmc_psrq   | PMC→pmc，评审日期→psrq |
| 创建时间       | cjsj       | 创建→cj，时间→sj |
| 项目名称       | xmmc       | 项目→xm，名称→mc |
| 姓名           | xm         | 姓→x，名→m |
| 工号           | gh         | 工→g，号→h |

### 重复标记规则

- **【重复】列**：根据【拼音简称】是否在整表中重复填写。
  - 若该行的拼音简称在表中出现**多于一次**（即存在其他行的拼音简称与之相同），则【重复】为 **「是」**。
  - 若该行的拼音简称在表中**仅出现一次**，则【重复】为 **「否」**。
- **示例**：工单→gd、高度→gd，两行拼音简称均为 `gd`，故两行【重复】均为「是」；创建时间→cjsj 唯一，【重复】为「否」。

## ⚠️ 注意事项

### 数据要求

1. **必需列**：必须存在【字段名称】列，否则规则报错并提示缺少该列。
2. **空值**：若某行【字段名称】为空，【拼音简称】建议为空字符串或由实现约定（如填空、或保留为 NaN）。
3. **编码**：Excel 与源数据建议使用 UTF-8 兼容编码，避免中文乱码。

### 转换说明

1. **多音字**：按通用拼音库的默认读音取首字母，不做多音字选择。
2. **特殊符号**：除下划线用于分段外，其他符号（如空格、横线）是否参与分段或保留，可在实现时在 py 中统一约定，并在本文档中补充说明。
3. **仅下划线**：若某段仅为 `_` 或空，可输出为空或单个下划线，以 py 实现为准。

### 错误处理

- 缺少【字段名称】列：返回明确错误信息，不写入结果。
- 读取/写入 Excel 异常：捕获并返回错误信息。

### 性能与依赖

- 依赖拼音库（如 `pypinyin`），需在 `requirements.txt` 或环境中安装。
- 实现中对**单字拼音首字母**做了缓存，相同汉字只调用一次拼音库，可显著提升多行、多重复字时的处理速度。

## 🔧 技术实现要点

### 建议流程

1. 读取当前工作表为 `DataFrame`，检查是否存在【字段名称】列。
2. 对【字段名称】列逐行转换（单字拼音首字母使用缓存）：
   - 按 `_` 分段；
   - 每段按「英文/数字小写 + 中文拼音首字母」规则得到子串；
   - 用 `_` 连接得到该行拼音简称。
3. 将结果写入新列【拼音简称】。
4. 根据【拼音简称】列使用 `duplicated(keep=False)` 判断重复，写入新列【重复】（重复为「是」，否则「否」）。
5. 返回带【拼音简称】与【重复】的 `DataFrame`，由主程序写入「结果」工作表。

### 关键依赖与函数（py 实现时）

- 拼音转换：使用 `pypinyin`（如 `lazy_pinyin(..., style=pypinyin.Style.FIRST_LETTER)` 取首字母），并对单字结果做缓存以提速。
- 主入口：`process(data_df, **kwargs)`，返回 `pd.DataFrame`（含原表列 + 【拼音简称】+ 【重复】）。
