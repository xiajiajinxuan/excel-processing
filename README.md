# 待办事项管理器

这是一个简单的待办事项管理桌面应用程序，使用Python和Tkinter开发。

## 功能特点

- 添加新的待办事项
- 删除已有的待办事项
- 标记待办事项为已完成/未完成
- 自动保存数据到本地文件
- 简洁美观的用户界面

## 安装与运行

### 环境要求

- Python 3.6 或更高版本

### 安装步骤

1. 克隆或下载此仓库到本地
2. 安装依赖项（可选）：
   ```
   pip install -r requirements.txt
   ```
3. 运行应用程序：
   ```
   python main.py
   ```

## 使用说明

- **添加任务**：点击"添加任务"按钮，在弹出的对话框中输入任务内容
- **删除任务**：选择一个任务，然后点击"删除任务"按钮
- **标记完成**：选择一个任务，然后点击"标记完成"按钮（再次点击可取消标记）

## 数据存储

所有待办事项数据将保存在应用程序目录下的`todos.json`文件中。

## 打包为可执行文件

如果需要将应用程序打包为独立的可执行文件，可以使用PyInstaller：

```
pyinstaller --onefile --windowed main.py
```

打包后的可执行文件将在`dist`目录中生成。
